# NaintaraOpticals
A full-stack e-commerce web application for an opticals (eyewear) shop, built with Django. The platform allows customers to browse eyewear products, manage a cart, and place orders, while giving admins full control over inventory through a custom admin dashboard.

✨ Features


User Authentication — Sign up, login, logout, and session-based user management
Product Catalog — Browse eyewear products with categories, images, and pricing
Search Bar — Quickly search for products by name/keyword
Filters — Filter products by price range and goggle type
Wishlist — Save favorite products to a wishlist for later
Shopping Cart — Add, update, and remove items from cart
Order History — Users can view their past orders after checkout
Admin Dashboard — Manage products, inventory, and orders via Django Admin
Responsive UI — Built with HTML, CSS, and JavaScript for a clean shopping experience
